##  Описание
Этот проект — система обмена сообщениями на **FastAPI** с **WebSocket**.  

##  Функциональность
1. **Личные и групповые** чаты  
2. **Сохранение истории** чатов в базе данных  
3. **Темная тема**  
4. **Редактирование профиля**  

## Технологии
> **FastAPI** — серверный фреймворк
> 
> **WebSocket** — двухсторонее соеденение
> 
> **SQLite** — база данных

## Принципы работы
> в процессе написания...

## Планируется
1. Цензура
2. Поддержка отправки медиа файлов
3. Уведомления
4. Смена СУБД на PostgreSQL
   
## Установка и запуск
1. **Клонируем репозиторий**
   ```bash
   git clone https://github.com/N1KTO4ME/messanger.git
   ```
2. **Устанавливаем зависимости**
   ```bash
   pip install -r requirements.txt
   ```
3. **Запускаем проект**
   ```bash
   uvicorn app.main:app --reload
   ```
